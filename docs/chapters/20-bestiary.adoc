= Bestiary

Monsters use the same rules as heroes — attributes, skills, and W/S/S damage.

== Reading a Stat Block
[%header,cols="1,3"]
|===
| *Name* | Creature type and challenge
| *Attributes* | B/F/A/G/K/R modifiers
| *HP* | Health points | *DR* | Damage reduction
| *Attacks* | W/S/S damage per attack
| *Abilities* | Special powers
| *Dice* | Dice types the creature uses
|===

== Beasts
*Wolf:* HP 8, DR 0. Bite: 1d6/1d8/1d10. Pack Tactics: +1 when ally adjacent.
*Bear:* HP 20, DR 2. Claw: 1d8/2d8/3d8. Bite: 1d10/2d10/3d10.

== Undead
*Skeleton:* HP 5, DR 0. Sword: 1d6/1d8/1d10. Vulnerable: Bludgeoning (damage tier +1).
*Wraith:* HP 12, DR 3 (non-magical). Life Drain: 1d6/2d6/3d6 necrotic. Incorporeal: half damage from physical.

== Dragons
*Young Red Dragon:* HP 40, DR 4. Bite: 2d8/3d8/4d8. Fire Breath: 4d6/6d6/8d6 (30 ft cone). Dice: 3 Fire.

== Creating Monsters
Assign attributes (-2 to +2), give 1-3 attacks with W/S/S damage, add 1-2 abilities. HP = 5 × desired difficulty (1-10).
