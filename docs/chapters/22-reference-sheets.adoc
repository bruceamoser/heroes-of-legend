= Quick Reference Sheets

== Core Resolution
| *3d6 + Attribute + Skill + Modifiers* |
| Weak (1-6): Partial / Standard (7-12): Full / Strong (13+): Exceptional |
| Crit (3×6): Auto-Strong + bonus / Fumble (3×1): Auto-fail + complication |

== Actions in Combat
| Standard: Attack, cast, dash, help | Move: Speed, stand | Bonus: Class feature |
| Reaction: Shield block, opportunity | Free: Talk, gesture |

== Conditions Quick-Ref
Blinded (tier -1), Charmed (can't attack), Frightened (can't approach), Grappled (speed 0), Incapacitated (no actions), Invisible (can't target), Paralyzed (auto-crit on hit), Poisoned (disadv attacks), Prone (stand=move), Restrained (speed 0, disadv), Stunned (no actions, no move), Unconscious (incap+prone)

== Damage Types
Physical: Slashing, Piercing, Bludgeoning | Elemental: Fire, Cold, Lightning, Acid, Poison | Magical: Force, Radiant, Necrotic, Psychic

== Dice Type Catalog
Elemental: Fire, Earth, Wind, Water | Weapon: Blades, Axes, Polearms, Great Weapons, Archery | Defense: Protection, Armor | Primal: Animal | Arcane: Energy
