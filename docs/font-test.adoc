== Side-by-Side Summary

[cols="1,3a", options="header"]
|===
| Font | Best Role
| Liberation Serif | Baseline — clean but generic
| EB Garamond | *Body text* — classic literary fantasy feel
| Cardo | Body text — scholarly, warm, distinctive serifs
| Linux Libertine | Body text — zero-dependency (pre-installed on Linux)
| Almendra | *Headings* — medieval fantasy display, pairs with EB Garamond
| Pirata One | *Chapter titles only* — gothic blackletter, use sparingly
|===

[NOTE]
====
*Crimson Pro* and *Alegreya* (also recommended) are variable-font-only on
Google Fonts and require the static TTF export from https://fonts.google.com —
download the family ZIP, extract, and add to the theme catalog.
====
